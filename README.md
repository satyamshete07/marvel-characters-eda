# Marvel Characters EDA Analysis

## 📊 Project Overview
This project performs Exploratory Data Analysis (EDA) on Marvel characters dataset to discover patterns, trends, and insights.

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 🔍 Key Steps
- Data Cleaning (handling missing values, duplicates)
- Outlier Detection
- Data Visualization (8+ graphs including pie chart)
- Statistical Analysis

## 📈 Insights
- Distribution of characters based on alignment
- Popularity trends
- Correlation between features

## 📁 Files Included
- Dataset (.csv)
- Jupyter Notebook (.ipynb)
- Presentation (.pptx)

## 🚀 Conclusion
EDA helps in understanding data patterns and making data-driven decisions.
